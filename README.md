# Networking - Från brandvägg till virtuella maskiner via VLAN

Praktik på Akind, målet vara att från ISPs IP-Nummer (Default Gateway hos Akind) skarva access till en webbserver huserandes hos en virtuell Ubuntu - Server i Proxmox.

# AI - Computer Vision
The project leverages the YOLOv4 algorithm to detect and track speed cameras on Swedish roads, utilizing a dataset of over 2,000 annotated images. The algorithm is capable of detecting objects from a distance of 200 to 250 meters.
